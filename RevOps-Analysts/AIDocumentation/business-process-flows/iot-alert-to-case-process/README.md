# Business Process Flow: IoT Alert to Case Process

## Overview
Guides users through a structured process on the msdyn_iotalert table.

## Stages (5)
- Stage -1: Create Case (incident)
- Stage -1: Created (msdyn_iotalert)
- Stage 4: Identify (incident)
- Stage 5: Research (incident)
- Stage 6: Resolve (incident)

## Primary Entity
msdyn_iotalert

## Description
Use this process flow to convert an IoT alert into a customer service case.
