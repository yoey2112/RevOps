TITLE
=====
RevOps - Power Pages Overview

PURPOSE
-------
This folder contains documentation for all Power Pages sites, web pages, and portal components in the RevOps environment.

Audience:
- RevOps developers and solution architects
- Copilot agents performing analysis and impact assessment
- Business analysts understanding system capabilities

SCOPE
-----
Each Power Pages component has its own subfolder containing:

- README.md: Component purpose and configuration
- facts/: Structured metadata specific to the component type

REVOPS STANDARD
---------------
- All documentation follows a consistent structure with _facts folders for structured data
- Dependency relationships tracked in _graph for impact analysis
- Asset registry in _registry catalogs all components
- Extraction runs logged in RUNS folder with timestamps and coverage reports

HOW TO NAVIGATE
---------------
To find specific Power Pages components:
- Browse subfolders by component type and name

To understand portal configuration:
- Review component README.md files for context
- Check facts/ folders for structured metadata
